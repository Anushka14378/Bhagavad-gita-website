# 🕉️ Bhagavad Gita Web Project

## 📚 About the Project

This project presents the **Bhagavad Gita**, one of the most important spiritual texts of India, in a simple and digital format. It helps users read and understand the teachings of Lord Krishna in an easy way.

---

## 🌟 What You’ll Find Here

* 📖 Complete chapters of Bhagavad Gita
* 🧾 Verses with clear structure
* 🎨 Clean and simple design
* ⚡ Fast and lightweight website

---

## 🧑‍💻 Built With

* HTML5
* CSS3
* JavaScript

---

## ▶️ Getting Started

To run this project locally:

* Clone the repository
* Open the folder
* Run `index.html` in your browser

---

## 🎯 Objective

The main aim of this project is to spread the knowledge of Bhagavad Gita using modern web technology so that anyone can access it anytime.
